# Hash（雜湊）

## 什麼是 Hash

一句話：把東西變成可以 O(1) 找到的鍵值索引工具

就像生活中的例子：
- 電視節目用頻道編碼，就能直接跳
- 通訊錄用名字查電話，而不是從頭找一遍

## C++ 常用容器

```cpp
unordered_map<Key, Value> mp;  // 鍵值對映射
unordered_set<Key> st;         // 唯一值集合
```

時間複雜度特性：

```txt
平均情況通常假設哈希函數良好且碰撞少。
最壞情況發生在：
所有元素哈希到同一個桶 → 哈希表退化為鏈表。
rehash 發生 → 需要重新哈希全部元素。
空間複雜度：O(N)，與元素數量成正比。
```

## 為什麼這麼重要

90% 的計算機問題本質就是「查找、計數、判斷是否存在」

Hash 可以幫你快速做這些事：
- 記錄次數
- 判斷出現過沒有
- 去重
- 儲存狀態

幾乎所有這些類型都會用到：字串問題、陣列問題、雙人互動、統計數據

## 常用模板

### 計數地圖

最常用的模板：

```cpp
unordered_map<int, int> mp;
for (int x : nums) {
    mp[x]++;
}
```

### 判斷是否出現過

```cpp
unordered_set<int> st;
if (st.count(x)) {
    // 出現過
}
st.insert(x);
```

### 字符頻率統計

常用於字串問題：

```cpp
vector<int> cnt(26, 0);   
for (char c : s) {
    cnt[c - 'a']++;
}
```

### 雙 key 處理

處理二維座標或配對：

```cpp
unordered_map<long long, int> mp;
long long key = (long long)x << 32 | y;  // 結合兩個 int
mp[key]++;
```

## 實戰應用

常見題型：
1. Two Sum - 找兩數之和
2. 字串異位詞 - 字符頻率統計
3. 最長不重複子串 - 滑動窗口配合 Hash
4. 群組異位詞 - 排序後作為 key

注意事項：
- Hash 表不保證順序
- 最壞情況可能退化為 O(N)
- 空間換時間的典型例子

Hash 是解決查找問題的神器，掌握好這些模板就能應對大部分情況。
