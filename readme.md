## 維護與更新

### 版本命名
採用日期版本號：`YYYY.MM.DD`

### 版本更新流程
1. 更新 `assets/js/script.js` 中的 `CURRENT_VERSION`
2. 提交到 GitHub
3. 用戶可以透過側邊欄的「版本檢查」按鈕手動檢查更新
