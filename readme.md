## 快速更新資源
當網站更新後，瀏覽器可能會使用舊的快取內容。使用以下方法立即獲取最新資源：

### 方法一：版本檢查功能（推薦）
在瀏覽器控制台（F12）輸入：
```javascript
checkVersion()
```
這會自動清除快取並重新載入最新資源。

### 方法二：手動清除快取
- **Windows**: `Ctrl + Shift + R` 或 `Ctrl + F5`
- **Mac**: `Cmd + Shift + R`

## 維護與更新
### 版本命名
採用日期版本號：`YYYY.MM.DD`

### 版本更新流程
1. 更新 `assets/js/script.js` 中的 `CURRENT_VERSION`
2. 提交到 GitHub
3. 用戶訪問網站時會自動檢測版本並提示更新
